Ce module permet à un administrateur ou un opérateur technique de récolter localement les défauts et dysfonctionnements de la plate-forme. Il peut être utilisé dans le cadre de l'exploitation de Moodle, mais également comme outil de résolution de défaut dans le cas général. Il peut être instancié plusieurs fois dans le même cours comme un module d'activité.

    La fiche de description de défaut est paramétrable. Il est possible de définir les rubriques que l'on souhaite faire détailler par les utilisateurs. Le moteur de recherche intégré tient compte de ce paramétrage.
